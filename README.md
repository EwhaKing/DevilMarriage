# 악마와의 계약은 혼인신고로! (DevilMarriage)
**2026 KING CRAFT 1팀**

## 프로젝트 소개
<img width="1223" height="638" alt="Image" src="https://github.com/user-attachments/assets/1389196b-a2bc-4a60-b0d0-9ab07826f285" />
악마 신화에 심취한 주인공이 왕국의 금서관에서 몰래 대악마를 소환한 뒤 결혼하기 위해, 마법진을 한붓그리기 퍼즐처럼 완성하고 다양한 악마들을 수집하고 사육하는 병맛 오컬트 퍼즐 게임.

## 게임 정보

| 항목 | 내용 |
|------|------|
| 장르 | 퍼즐 |
| 엔진 | Unity 2D 6000.4.1f1 |
| 해상도 | 가로 |
| 조작법 | PC 키보드 |
| 실행 | [플레이 유니티](https://play.unity.com/ko) |

## 주요 기능

### 🧠 정신력 시스템
마법진 오류(잘못된 획, 실행 취소, 의도치 않은 개체 소환) 발생 시 정신력이 감소하며, 정신력이 모두 소진되면 게임 오버.

### 🔯 마법진 시스템
마법진을 그려 악마를 소환.

### 🩸 쥐의 피
- 마법진을 그릴 때 필요한 자원
- 양이 한정되어 있어 최소한의 획으로 마법진을 그려야 함
- 피가 부족할 경우 실행 취소(정신력 소모)로 되돌린 뒤 다시 채워야 함
